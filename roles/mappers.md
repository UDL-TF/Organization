# Mappers

## Role Description

Mappers design and build custom Team Fortress 2 maps that keep gameplay fresh, align with community feedback, and support server events.

## Responsibilities

- Prototype, iterate, and ship custom TF2 map layouts
- Collaborate with Developers and Server Managers to ensure technical compatibility and deployment readiness
- Gather player feedback with Community Managers and adjust map flows accordingly
- Maintain documentation for asset requirements, version history, and deployment procedures
